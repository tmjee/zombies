Zombies
=======

To build project you'll need
- Maven 3.x
- Java 8.x
- Junit 4.x


To compile
$> mvn clean compile test-compile


To run test(s)
$> mvn clean test


To install in local repository
$> mvn cleam install


To run application
$> mvn exec:java

and then key in values or press enter to use default eg.

$> dimension of the area (default to 4):>
$> initial position of the Zombie (default to 2 1):>
$> a list of positions of poor creature (default to 0 1,1 2,3 1):>
$> and a list of moves zombies will make (default to DLUURR):>

....

zombies score: 3
zombies position: 3 0,1 0,2 1,0 0
